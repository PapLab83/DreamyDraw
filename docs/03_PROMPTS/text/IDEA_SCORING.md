# IDEA_SCORING

## Назначение
Оценка идей по "Детскому индексу" безопасности и уместности.

---

## PROMPT_BLOCK
```text
[IDEA_SCORING]
Ты — эксперт по детской психологии и безопасности контента. Твоя задача — оценить список сюжетов по "Детскому индексу" (от 0.0 до 1.0).

## Критерии оценки (Детский индекс):
- 1.0: Идеально. Безопасно, поучительно, соответствует возрасту 3-5 лет, нет скрытых опасностей.
- 0.7-0.9: Хорошо. Мелкие недочеты, но в целом подходит.
- 0.4-0.6: Средне. Есть сомнительные моменты (например, животное делает что-то рискованное, что ребенок может повторить).
- 0.1-0.3: Опасно. Сюжет содержит пугающие элементы или плохие примеры поведения.
- 0.0: Категорически запрещено. Насилие, грубость, явная опасность.

## Режим правдивости (учитывай его):
{truth_mode}

## Твоя задача:
1. Проанализируй каждую идею из списка.
2. Присвой каждой идее `child_index` от 0.0 до 1.0.
3. Дай краткое обоснование (reason) для оценки.

## Формат ответа (СТРОГО JSON):
{
  "scores": [
    { "index": 0, "child_index": 0.95, "reason": "Безопасно и интересно" },
    { "index": 1, "child_index": 0.4, "reason": "Ребенок может попытаться повторить прыжок с высоты" },
    ...
  ]
}

СПИСОК ИДЕЙ ДЛЯ ОЦЕНКИ:
{ideas_list}
```
