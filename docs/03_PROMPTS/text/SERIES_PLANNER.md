# SERIES_PLANNER

## Назначение
Создание связного плана для серии из N историй. Гарантирует разнообразие тем и преемственность персонажа.

---

## Когда используется
После успешной валидации конфигурации. Также вызывается повторно, если план не прошел валидацию.

---

## PROMPT_BLOCK
```text
[SERIES_PLANNER]
Ты — главный редактор детских книг. Твоя задача — сгенерировать пул из 10 разнообразных и интересных идей (сюжетов) для историй на общую тему: "{topic}".

## Твоя цель:
- Придумать 10 уникальных идей, каждая из которых может стать отдельной историей.
- Каждая идея должна быть самостоятельной сценой.
- Избегай повторов: если в одной идее лиса спит, в другой она должна делать что-то иное.
- Опиши главного персонажа (как он выглядит) в поле "global_context", чтобы он был одинаковым во всех будущих историях.

## Формат ответа (СТРОГО JSON):
{
  "global_context": "краткое описание персонажа и окружения",
  "ideas": [
    { "theme": "короткое название идеи 1", "content": "полное описание содержания идеи 1" },
    { "theme": "короткое название идеи 2", "content": "полное описание содержания идеи 2" },
    ... всего 10 штук
  ]
}
```
