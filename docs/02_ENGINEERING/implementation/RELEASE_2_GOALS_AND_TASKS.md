Первые ШАГИ плана (как бы я рекомендовал двигаться):

- фиксируем параметры (количество историй, режим правды, национальность) как жестко заданные
- берем кейс сказочной.лисички с собираем промпт который получает агент для просто сказочной лисы, для лисы в русско-народном стиле, для лисы в стиле чуковского. Нужны именно полные промпты для того чтобы провести анализ. 
- по результатам анализа промпта скорее всего можно будет увидеть следующие проблемы
  - много технической терминологии
  - много неестественных фраз
  - архитектурные промахи при сборки промпта из несогласованности разных частей
- предложить решения выявленных проблем, например:
  - очистить промпт генератора от технических деталей - их может стоит передать на следующий этап работы - если надо разделить работы, чтобы промпт для генерации текста был максимально чистым
  - сделать части тела промпта из разных источников более согласованными
  - определить порядок добавления разных частей промпта 
  - сделать части промпта независимыми
- ожидать что по результатам изменений результат генерации текста станет не просто адекватным но интересным 
- после получения результатов с промптами потребуется обновить код и документацию проекта:
  - для начала определим перечень обновлений
  - согласуем перечень
  - внесем обновления 
  - проведем тесты 
  - вероятно придется разделять код сборки промпта для сказок и других вариантов режима правды как временный костыль релиза 2 - его надо обязательно будет закрыть в конце перед сдачей релиза - для этого всегда должен быть этап клинап
- после обновления кейса сказочной лисицы с разных стилях можно опционально подготовить промпты для других сказочных животных. я бы рекомендовал хотя бы еще одно-два животных добавить чтобы обкатать независимость субстилей
- далее можно переходить к этапу с кейсами в режиме поучительной правды для лисы и проделывать такую же работу только со стилями юного натуралиста и энциклопедического. В идеале если возможно структуру сборки промпта следует сделать такой же как и для лисы в сказке. тогда мы остальных животных тоже быстро адаптируем
