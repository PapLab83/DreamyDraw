# Приложение: Обучение английскому языку (English with Dreamy)

Статус: future concept / Release 2+ candidate. Это не фактическая функциональность Release 1. Полезные идеи из этого приложения должны быть перенесены в `docs/02_ENGINEERING/implementation/RELEASE_2_BACKLOG.md` или отдельный product brief перед удалением/архивацией документа из активной навигации.

## Концепция
Превращение генератора историй в инструмент мягкого погружения в английский язык для детей 3-5 лет. Использование визуальных образов для закрепления лексики.

## Механика реализации

### 1. Текстовая генерация (LLM)
- **Смешанный текст:** История пишется на русском, но целевые объекты (животные, цвета, действия) вводятся на английском.
- **Пример:** "Маленький **Red Fox** (рыжий лис) нашел большое **Apple** (яблоко). Он очень **Happy** (счастлив)!"
- **Транскрипция:** Для родителей добавляется подсказка по произношению в скобках.

### 2. Визуальное подкрепление (Image Generation)
- **Промпт:** Инструкция генератору включает требование нарисовать объект четко и добавить текстовую метку на английском языке прямо на картинку (инфографический стиль для детей).
- **Фокус:** Если изучаем слово "Fox", лиса должна быть центральным и самым ярким объектом.

### 3. Интерактив (Вопросы)
- Вопросы направлены на повторение и узнавание:
    - "Какого цвета был лис? (Red)"
    - "Что нашел лис на картинке? (Apple)"
    - "Повтори вместе с мамой: Happy Fox!"

## Техническое расширение (Масштабируемость)
- **Новый провайдер видео:** Интеграция с API (Runway/Sora) для создания 5-секундных анимаций, где объект совершает действие, связанное с изученным словом.
- **Аудио-поддержка:** Добавление TTS (Text-to-Speech) для озвучивания английских фраз с правильным акцентом.

## Почему это эффективно?
Для ребенка 3-5 лет обучение происходит через игру и визуальные ассоциации. DreamyDraw создает персональную историю, где ребенок становится участником процесса, что значительно повышает запоминаемость слов по сравнению со статичными карточками.
